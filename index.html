About Venom

Venom is a comprehensive social networking platform designed specifically for college students. It provides an intuitive and secure digital space for students to connect, share, and engage with their peers.

Key Features

  • Secure User Authentication
  • Profile Customization
  • Meme Sharing Platform
  • Anonymous Confessions
  • Q&A Section
  • Real-time Chat
  • Social Matching

Technology Stack

  • React Frontend
  • Express Backend
  • PostgreSQL Database
  • WebSocket for Real-time Features
  • Tailwind CSS for Styling
  • Drizzle ORM
